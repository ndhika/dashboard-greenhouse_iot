# 🌱 Dashboard Greenhouse IoT

Sebuah aplikasi antarmuka web (dashboard) untuk memonitor dan mengelola kondisi lingkungan *greenhouse* secara *real-time* berbasis *Internet of Things* (IoT). Proyek ini dirancang untuk memudahkan pemantauan suhu, kelembaban, dan parameter penting lainnya agar tanaman dapat tumbuh dalam kondisi yang paling optimal.

## 🚀 Fitur Utama

* **Real-time Monitoring:** Memantau metrik langsung dari sensor IoT (seperti Suhu, Kelembaban Udara, dan Kelembaban Tanah).
* **Data Visualization:** Menampilkan riwayat data dalam bentuk grafik interaktif yang mudah dianalisis.
* **Status Perangkat:** Memantau status konektivitas perangkat keras IoT (Online/Offline) secara langsung.
* **Responsive Design:** Antarmuka yang rapi dan responsif, nyaman digunakan baik melalui perangkat desktop maupun *mobile*.

## 🛠️ Teknologi yang Digunakan

* **Frontend:** TypeScript, React (Vite)
* **Backend/Database:** Supabase / Node.js
* **IoT & Perangkat Keras:** C++ (untuk mikrokontroler seperti ESP32/ESP8266)
* **Deployment:** Vercel

## ⚙️ Persyaratan (Prerequisites)

Pastikan sistem Anda telah memiliki perangkat lunak berikut sebelum menjalankan proyek ini secara lokal:

* [Node.js](https://nodejs.org/) (Versi 16 atau yang lebih baru)
* NPM, Yarn, atau pnpm

## 💻 Cara Instalasi (Local Development)

Ikuti langkah-langkah berikut untuk menjalankan dashboard ini di komputer Anda:

1. **Clone repositori ini:**
   ```bash
   git clone [https://github.com/ndhika/dashboard-greenhouse_iot.git](https://github.com/ndhika/dashboard-greenhouse_iot.git)
   cd dashboard-greenhouse_iot
Instal dependensi:
   ```Bash
   npm install
  ```

# atau jika menggunakan yarn:
   ```Bash
   yarn install
   ```
   ```Bash
   npm run dev
   ```
# atau:
   ```Bash
   yarn dev
   ```
